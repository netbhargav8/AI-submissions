# MNIST Handwritten Digit Classifier

A computer vision classification engine that flattens $28 \times 28$ pixel matrices and employs a multi-class Logistic Regression model to recognize and classify handwritten digits (0-9).

## Prerequisites & Installation

This project requires Python 3, Scikit-Learn, NumPy, and Matplotlib. Install the execution dependencies using your terminal:

```bash
pip install scikit-learn numpy matplotlib
